# Prof. Wadii Boulila — Academic Portfolio

A professional academic portfolio website built for GitHub Pages.

## 🚀 Quick Deploy

1. Fork or clone this repository
2. Go to **Settings → Pages → Source → Deploy from a branch** → select `main` / `root`
3. Your site will be live at `https://yourusername.github.io/repository-name/`

## 📁 Structure

```
├── index.html          # Main landing page (single-file, self-contained)
├── publications.html   # Full publications page
├── README.md           # This file
└── CNAME              # (Optional) Custom domain config
```

## ✏️ Customization

- **Citation metrics**: Search for `[XX]` in `index.html` and replace with your live h-index / i10-index
- **Profile links**: Update Google Scholar, Scopus, ORCID, and LinkedIn URLs in the hero section
- **Photo**: Add a profile photo by replacing the hero section header
- **Colors**: Modify CSS variables in `:root` at the top of the `<style>` block
- **Content**: All content is in plain HTML — edit sections directly

## 🎨 Design

- **Aesthetic**: Dark academic luxury — deep navy, warm gold accents, Cormorant Garamond + DM Sans
- **Features**: Scroll-triggered animations, animated counters, responsive mobile menu, smooth navigation
- **Performance**: Single HTML file, no build step, CDN-loaded fonts/icons only
